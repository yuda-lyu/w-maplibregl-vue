WMaplibreglVue

WMaplibreglVue2 - 使用 MapLibre GL JS 重構的地圖組件（Vue 2） 所有面板透過四角容器 (corner container) 自動堆疊，不重疊。

核心業務邏輯已提取至 src/js/ 模組，此元件僅保留：

  • 響應式狀態（data / computed / watch）
  • 薄委派方法（調用模組函式 + 同步 Vue 狀態）
  • Vue 特定操作（$nextTick / $refs / $forceUpdate）
